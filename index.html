Weather Dashboard

The Weather Dashboard is an interactive JavaScript web application that fetches real-time weather data from an external API and presents it in a clean and user-friendly interface. Users can search for any city worldwide to view current weather conditions, temperature trends, humidity levels, and a 5-day forecast. The application features dynamic charts, responsive layout, and error handling for invalid searches. Built using HTML, CSS, and JavaScript, this project demonstrates API integration, asynchronous data fetching with fetch(), DOM manipulation, and responsive design principles, making it both visually appealing and functional across devices.
